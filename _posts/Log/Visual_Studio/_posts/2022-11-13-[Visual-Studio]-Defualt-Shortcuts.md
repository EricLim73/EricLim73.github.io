# Default Shortcut 


## ShortCuts for Visual Studio		

| Shortcut(default)                 | What it does                      |
|:----------------------------------|:----------------------------------|
| Ctrl + k, c                       | Commenting Line                   |
| Ctrl + k. u                       | Undo Comment                      |
| Alt Drag                          | Select Area                       |
| F12 while selecting the target    | Jump to Declaration / Definition  |

## Debug
| Shortcut(default)         | What it does                  |
|:--------------------------|:------------------------------|
| F5                        | Start Debug                   |
| F9                        | Set/undo BreakPoint           |
| F10                       | Debug 1 line                  |
| F11                       | Debug step in function        |
| Shift + F5                | Stop Debugging                |
| F5 again while debugging  | Step to next bearkPoint       |
